🚀 QUICK START (5 MINUTES)
================================================================================

1. Install Dependencies:
   pip install -r requirements.txt

2. Configure Database (edit core/settings.py):
   DATABASES = {
       'default': {
           'ENGINE': 'django.db.backends.postgresql',
           'NAME': 'bursary',
           'USER': 'postgres',
           'PASSWORD': 'your_password',
           'HOST': 'localhost',
           'PORT': '5432',
       }
   }

3. Run Migrations:
   python manage.py migrate
   python manage.py createsuperuser

4. Start Server:
   python manage.py runserver

5. Access System:
   Admin: http://localhost:8000/admin/
   API: http://localhost:8000/bursary/

📊 API ENDPOINTS
================================================================================

Public Endpoints:
POST   /bursary/apply/                    - Submit application
GET    /bursary/deadline/                 - Check deadline status

Admin Endpoints:
GET    /bursary/applications/             - List applications (paginated)
GET    /bursary/applications/<ref>/       - Get specific application
PUT    /bursary/applications/<ref>/update-status/  - Update status
GET    /bursary/applications/<ref>/history/       - View status history

Authentication:
POST   /bursary/logout/                   - Logout
